# 物件リストV2 → 配布完了アーカイブ 移管

物件リストV2 でステータスが「配布完了」になったレコードを、配布完了確認シートの `配布完了` タブに移管する GAS。

## 対象

- **ソース**: 物件リストV2（スプレッドシート・タブ名は要確認）
- **先**: [配布完了確認シート](https://docs.google.com/spreadsheets/d/1wIE_FrIv4a7QoeMcKROAYesxbMIFmsHwAXFV6k-6h0Y) の `配布完了` タブ

## トリガー

**要確認**: スプレッドシートの Apps Script で、以下のいずれかが設定されている想定。

- **時間主導型**: 例）毎日 0:00、毎時 0 分 など
- **編集時**: 物件リストV2 の編集時に実行
- **手動**: 必要に応じて手動実行

### 確認手順

1. 物件リストV2 または 配布完了確認シート を開く
2. **拡張機能** → **Apps Script**
3. 左サイドバー **トリガー**（時計アイコン）
4. 登録済みトリガーを確認

## コード

`Code.gs` に実装を格納。GAS エディタからコピーしてこのリポジトリに追加してください。

## 注意

- 物件リストV2 のスプレッドシート ID・タブ名はコード内で要設定
- 移管後のソース側の扱い（削除/ステータス変更）は実装次第
