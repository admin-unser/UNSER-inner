# Google Apps Script (GAS)

このディレクトリには、UNSER 社内で使用する GAS のコードをバージョン管理用に格納します。

**注意**: GAS は Google の環境で実行されます。このリポジトリのコードは参照・バックアップ用です。実際の実行はスプレッドシートに紐づいた GAS プロジェクトで行われます。

## 優先度

- **優先度1**: 社内管理・レポート作成（このリポジトリの Python スクリプト）
- **優先度2**: GAS のコード管理・ドキュメント化

## トリガーについて

GAS のトリガーは **スプレッドシートの GAS プロジェクト** 内で設定します。このリポジトリにはトリガー設定は含まれません。

### トリガーの種類

| 種類 | 発生タイミング | 用途例 |
| --- | --- | --- |
| **時間主導型** | 分/時/日/週/月の指定時刻 | 定期移管、日次バッチ |
| **編集時** | スプレッドシートのセル編集時 | リアルタイム反映 |
| **フォーム送信時** | フォーム回答時 | 回答の自動処理 |
| **手動** | スクリプトエディタから実行 | テスト、単発実行 |

### トリガーの確認方法

1. スプレッドシートを開く
2. **拡張機能** → **Apps Script**
3. 左サイドバーの **トリガー**（時計アイコン）をクリック
4. 登録済みトリガー一覧で、関数名・実行タイミング・エラー通知先を確認

### トリガーの設定方法

1. Apps Script エディタで **トリガー** をクリック
2. **トリガーを追加**
3. 実行する関数、イベントの種類、時間間隔 などを指定

## コードの同期

### リポジトリ → GAS（手動）

1. このリポジトリの `gas/` 配下のコードをコピー
2. Apps Script エディタに貼り付け・保存

### GAS → リポジトリ（手動）

1. Apps Script エディタでコードをコピー
2. このリポジトリの `gas/` 配下に保存
3. `git add` → `git commit` → `git push`
